# 🔐 Password Strength Checker & Generator

## 📌 Overview

Password Strength Checker & Generator is a Python console application that analyzes the strength of a password and generates secure random passwords. The project is completely menu-driven and was built to strengthen core Python programming concepts.

---

## ✨ Features

* Check password strength
* Password score (0–5)
* Detect:

  * Minimum length (8 characters)
  * Uppercase letters
  * Lowercase letters
  * Digits
  * Special characters
* Suggest improvements for weak passwords
* Generate strong random passwords
* Menu-driven interface
* Automatically verify the generated password
